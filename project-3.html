SCI

The webiste is the official site of Student Code-in program started by our team in covid to helps students get started with open-source.

Live Link

What is the motivation behind?

The motivation behind building this website is to provide a platform for students to learn about open-source, and to promote the initiative of StudentCode-in. The website aims to provide students with a comprehensive overview of the program, upcoming events, and resources that can help them get started with open-source journey.

Project Overview

This website is an overview page for StudentCode-in, which is an initiative that aims to help students learn coding and computer science. It provides information about the program, upcoming events, and resources for students to get started with coding. It has a clean and simple design, with sections for introduction, events, and resources. The website also features links to the organization's social media accounts and contact information.

Problems occurred

Problems that I faced during this projects are:
In ensuring that the website is accessible for users with low-bandwidth connection.
In handling the website traffic.
In providing regular maintenance and security updates.
In ensuring that the website is optimized for search engine ranking.
